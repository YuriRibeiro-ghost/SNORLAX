Este é um exemplo de parágrafo (tag <p>). Ele serve para agrupar blocos de texto de forma organizada.
A tag <br> acima forçou essa frase a ir para a linha de baixo, mesmo estando dentro do mesmo parágrafo.

Podemos aplicar diferentes tipos de ênfase no texto:
O uso da tag <strong> indica que este texto é muito importante.
Já a tag <em> é usada para dar ênfase, deixando o texto em itálico. É ideal para termos técnicos ou estrangeiros.
